# 附录

## A. 常用命令速查
- `openclaw gateway status` - 检查网关状态
- `openclaw gateway start` - 启动网关
- `openclaw gateway stop` - 停止网关
- `openclaw gateway restart` - 重启网关

## B. 配置文件说明
- AGENTS.md - 代理配置
- SOUL.md - 人格设定
- USER.md - 用户信息
- TOOLS.md - 工具配置
- MEMORY.md - 长期记忆
- HEARTBEAT.md - 心跳任务

## C. 故障排除
### 常见问题及解决方案

1. **服务无法启动**
   - 检查端口占用情况
   - 查看日志文件
   - 验证配置文件语法

2. **工具调用失败**
   - 检查权限设置
   - 验证参数格式
   - 查看安全策略

3. **网络请求超时**
   - 检查网络连接
   - 验证API密钥
   - 调整超时设置
